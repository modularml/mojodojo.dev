# sort

Sorts a `DynamicVector[Int]` inplace, in ascending order


```mojo :no-line-numbers 
from Sort import sort
from Vector import DynamicVector

let v = DynamicVector[Int](3)

v.push_back(20)
v.push_back(10)
v.push_back(70)

sort(v)

for i in range(v.size):
    print(v[i])
```

    10
    20
    70
